✨ Características Implementadas:
🚀 Optimizaciones de Rendimiento:

Procesamiento paralelo con std::async y std::future

Operaciones asíncronas para I/O de archivos
Threads seguros con mutex
Medición de alta precisión con chrono
Gestión eficiente de memoria

🔐 Algoritmo de Encriptación:
Letras: Desplazamiento de 3 posiciones ASCII (A→D, B→E)
Números: Intercambio simétrico (0↔9, 1↔8, 2↔7)
Otros caracteres: Sin cambios

📊 Medición de Tiempos:
Tiempo de generación de copias
Tiempo de encriptación y hash
Tiempo de validación y desencriptación
Tiempo de comparación con original
Tiempo total (TTT) y tiempo promedio por archivo (TPPA)

🎯 Cómo Usar el Proyecto:
Opción 1: Compilación Rápida

# Ejecutar el script de compilación
compilar.bat

Opción 2: Dev C++

Abrir Dev C++
Crear nuevo proyecto C++
Copiar código de main_simple.cpp
Configurar: -std=c++11 -pthread
Compilar y ejecutar

Opción 3: Makefile

make           # Compilar
make run       # Compilar y ejecutar
make test      # Prueba automática con N=10
make clean     # Limpiar archivos

📈 Pruebas de Rendimiento:

El programa incluye un sistema de pruebas automáticas que:
Ejecuta el programa con diferentes valores de N (5, 10, 15, 20, 25)
Genera reportes de rendimiento automáticamente
Crea archivos CSV para análisis de datos
Compara tiempos y calcula mejoras de performance
